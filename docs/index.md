### **Summary**

A course where we learn how to :hammer_pick: Microeconomic models 

``` mermaid
flowchart LR
    A(Preferences/Income) -->  C(Demand);
    C --> E(Market Equilibrium);
    D(Supply) --> E
    M(Expectations) -->C;
    M --> D;
    Z(Taxes/Subsidies) --> C;
    Z --> D;
    A --> D;
    E --> Q(Quantity);
    E --> R(Prices);
```
